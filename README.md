# 小红书扫码发布 - 飞书多维表格字段捷径

一键生成小红书笔记发布二维码，扫码即可在小红书APP完成发布。

## 功能特性

- 📝 支持设置笔记标题（自动截取前20字）
- 📄 支持设置笔记正文内容
- 🖼️ 支持多图上传（首图为封面，其余为正文图片）
- 🏷️ 支持添加标签（自动解析 # 分隔的标签）
- 📱 生成二维码，扫码在小红书APP完成发布

## 使用方法

### 1. 添加字段捷径

在多维表格中添加「小红书扫码发布」字段捷径。

### 2. 配置字段映射

| 配置项 | 说明 | 必填 |
|--------|------|------|
| 标题字段 | 选择包含笔记标题的文本字段 | ✅ |
| 正文字段 | 选择包含笔记正文的文本字段 | ✅ |
| 图片字段 | 选择包含图片的附件字段 | ✅ |
| 标签字段 | 选择包含标签的文本/多选字段 | ❌ |

### 3. 触发生成

填写数据后，插件会自动生成发布二维码，使用小红书APP扫码即可完成发布。

## 字段输入格式

### 标题字段
普通文本，超过20字会自动截取。

### 正文字段
普通文本，支持换行。

### 图片字段
附件字段，支持多张图片：
- 第一张图片作为封面
- 其余图片作为正文配图
- 图片按文件名排序

### 标签字段
支持以下格式：
- 文本：`#标签1 #标签2 #标签3`
- 文本：`标签1,标签2,标签3`
- 多选字段：直接选择标签

## 返回结果

成功时返回小红书发布二维码图片（附件类型），用户扫码后在小红书APP完成发布。

## 错误处理

| 错误情况 | 处理方式 |
|----------|----------|
| 标题为空 | 返回错误，提示填写标题 |
| 正文为空 | 返回错误，提示填写正文 |
| 图片为空 | 返回错误，提示上传图片 |
| 网络异常 | 返回错误，提示稍后重试 |
| 额度用完 | 返回付费错误，提示购买 |
